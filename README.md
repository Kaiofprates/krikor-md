Este é um jogo de xadrez que você joga abrindo Issues! O tabuleiro é atualizado automaticamente a cada jogada.

Vez das: **Brancas**

## Como Jogar
Clique em um dos links abaixo para fazer sua jogada. Isso abrirá uma Issue pré-preenchida. Basta clicar em **"Submit new issue"** para confirmar o movimento.

### Movimentos Sugeridos (Abertura)

- [Nf3](https://github.com/YOUR_GITHUB_USERNAME/YOUR_REPO_NAME/issues/new?title=Chess+Move:+Nf3)

---
**Instruções de Configuração:**
1. Crie um repositório no GitHub.
2. Faça o push de todos os arquivos neste diretório para o repositório.
3. Edite os links acima no `README.md` (ou espere a primeira Action rodar para atualizar tudo automaticamente).
4. O GitHub Actions deve estar habilitado nas configurações do repositório (em "Actions" -> "General" -> "Read and write permissions").
